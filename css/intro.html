AT-rules

Sao regras realacionadas ao comportamento do CSS, começa com um sinal de@ seguido do identificador e do valor .

São os siguientes:

  • @import: serve para incluir um css externo.
    ex: @import url("http:linkexterno.com.uy/style.css")
  • @media: são regras condicionais para varios dispositivos.
    @media (min-width:365px){ }

  • @font-face: é para colocar fontes externas.
    @font-face{ }

  • @keyframes: serve para as animations do css.
    @keyframes animationName{ }
